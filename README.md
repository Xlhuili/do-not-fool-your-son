# 如何不让自己的孩子变傻

> **一本讲证据的育儿书。** 孩子不是变傻的，是被教傻的——"聪明"不是天生的分数，而是一组可培养、也可被日常做法系统性摧毁的心智能力。

## 这本书与众不同的地方

市面上的育儿建议，大量是没有出处的民间智慧。这本书给自己立了一条规矩：**每一条依赖外部事实的主张，都标注证据等级（A/B/C）**，争议处一律老实标 C、明确写出"科学还没定论"，**绝不把动听的说法伪装成科学结论**。

书里所有被奉为常识的"经典"——成长型思维、棉花糖实验、三千万词、一万小时、屏幕毁大脑——都经过逐条核查，该夸的夸、该拆的拆（详见 `19-附录B-重复性危机幸存者.md`）。每条主张的出处与等级，列在可逐条自查的源账本里（`18-附录A-源账本.md`）。

## 全书结构

- **导论** · 孩子不是变傻的
- **第一部分 机制**：重定义聪明 · 八问诊断 · 四种傻化机制 · 语言地基
- **第二部分 模式**：标准答案机器 · 多巴胺劫持 · 包办型瘫痪 · 恐惧驱动
- **第三部分 压力测试**：屏幕 · 抢跑 · 天赋/一万小时 · 诊断与标签
- **第四部分 现场指南**：环境设计 · 分龄指南 · 父母自己的心智 · 一页纸
- **后记** · 这本书是怎么写的
- **附录 A–E**：源账本 · 重复性危机幸存者 · 分龄速查卡 · 八问诊断卡 · 分级书目
