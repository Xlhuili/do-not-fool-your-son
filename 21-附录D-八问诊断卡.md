# 附录 D · 八问诊断卡

> 剪下来贴冰箱上。做任何一个关于孩子的决定前——报班、买平板、定规矩、安排周末、回应一次哭闹——过一遍。
> **用法：不求八问全过关，发现一两个红灯就停下来重想。**

---

```
┌─────────────────────────────────────────────┐
│        喂养，还是饿死？· 八问诊断              │
│                                               │
│  这件事，是在喂养还是在饿死孩子的心智？        │
│                                               │
│  1. 奖励"标准答案"，还是奖励"好问题"？       │
│  2. 制造即时快感，还是培养延迟满足？          │
│  3. 替他消除摩擦，还是让他在可承受的           │
│     摩擦里长本事？                            │
│  4. 动力来自恐惧/比较，还是好奇/兴趣？        │
│  5. 给思想提供了语言和材料吗？                │
│  6. 允许犯错并从中学习，还是惩罚错误本身？    │
│  7. 在喂养注意力，还是劫持注意力？            │
│  8. 把尺子从"成绩/听话"换成"长期心智能力"，  │
│     这个选择还成立吗？                        │
│                                               │
│  ▸ 不求全过关——一两个红灯，就停下来重想。    │
└─────────────────────────────────────────────┘
```

## 每一问对应它守护的地基

| 问 | 守护的心智地基 | 亮红灯多半牵连哪台机器 |
|---|---|---|
| 1 提问 vs 答案 | 好奇、思辨 | 标准答案机器 |
| 2 即时 vs 延迟 | 专注、自控 | 多巴胺劫持 |
| 3 摩擦 | 自主、韧性 | 包办型瘫痪 |
| 4 恐惧 vs 好奇 | 内在动机 | 恐惧驱动 |
| 5 语言材料 | 思辨的原料（语言地基） | — |
| 6 犯错 | 更新、韧性 | 恐惧驱动 |
| 7 注意力 | 专注 | 多巴胺劫持 |
| 8 换尺子 | 全部（压舱石问题） | 全部 |

## 一个示范：要不要买"学习平板"

- 问 1：点对得星星的标准答案模式 → ⚠️
- 问 2：即时音效、即时奖励、即时下一关 → ⚠️⚠️
- 问 7：不断变化的画面维持"注意" = 劫持 → ⚠️⚠️
- 问 8：换成"它在训练大脑偏好即时刺激还是深度专注"来衡量 → ⚠️

四问亮红灯，集中在专注和即时满足。结论不一定是"绝对不能碰"，但你已清楚**代价付在哪**——于是能做更精确的决定，而不是被"开发智力"四个字牵着走。

> 这就是诊断和建议的区别：建议让你将信将疑，诊断让你**自己看见代价付在哪**，并学会下次自己判断。
