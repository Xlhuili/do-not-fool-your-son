# 附录 C · 分龄"喂养 vs 饿死"速查卡

> 把第 13、14 章压缩成一张可打印的卡。找到你家孩子所在的年龄段，看这一段最该**喂养**什么、最该**饿死**（避免）什么、今天能做的**一件事**。
> 年龄是大致规律，不是打卡截止线——孩子各有节奏。

---

## 0–3 岁 · 地基的地基

| | |
|---|---|
| **这阶段在建什么** | 对世界的基本信任（安全依恋）+ 语言的爆发 |
| **最该喂养** | 可靠的回应（哭了有人来）· 真实的来回对话 · 自由的探索 |
| **最该饿死（避免）** | 用屏幕替代真人陪伴 · 因为他"还不懂"就不跟他说话 |
| **今天做一件事** | 换尿布、做饭、散步时，不停地跟他解说你在做什么、看到什么 |

## 3–6 岁 · 游戏就是工作

| | |
|---|---|
| **这阶段在建什么** | 想象力、语言、社会规则、自我控制——主要通过假装游戏 |
| **最该喂养** | 自由游戏的时间和空间 · 围绕绘本的来回对话 · 和同伴的相处 |
| **最该饿死（避免）** | 用提前的认字算术挤掉游戏 · "别人家都在学了"的抢跑焦虑 |
| **今天做一件事** | 给一段无安排、无屏幕的时间，扔几样开放式材料，然后**退后**让他自己玩 |

## 6–12 岁 · 胜任感的窗口

| | |
|---|---|
| **这阶段在建什么** | 胜任感 + 对学习的基本态度（有趣的探索，还是可怕的任务） |
| **最该喂养** | 内在动机（因为有趣而学）· "我自己搞定了"的真实体验 · 自己感兴趣的事 |
| **最该饿死（避免）** | 用"考不好就完了""你看别人家"驱动学习 · 把学习和恐惧焊死 |
| **今天做一件事** | 找一件他**自己**感兴趣的事（哪怕和成绩无关），认真支持他深入下去 |

## 青春期 · 从掌控到顾问

| | |
|---|---|
| **这阶段在建什么** | 自我认同 + 独立；大脑奖赏系统活跃、自控系统尚在成熟 |
| **最该喂养** | 真实的自主空间 · 不被冲突切断的亲子关系 · 被信任的感觉 |
| **最该饿死（避免）** | 把每件事都升级成控制权战争 · 用硬控代替放手 |
| **今天做一件事** | 把一个你一直替他做主的决定，正式交还给他，并说明"结果也你自己担" |

---

## 一条贯穿所有年龄的主线

> **你的角色，从"掌控者"一路退向"顾问"。**
> 0–3 岁近乎全面照料 → 6–12 岁是支持者和脚手架 → 青春期最好是被信任的顾问。
> 养育的成功，不是孩子一直需要你掌控，而是他**越来越不需要你掌控、却始终愿意和你保持连接**。
> 每个阶段适度、及时地放手，就是在为那个"能独立、又愿亲近"的最终关系铺路。

## 反向自检：我家在开哪台机器？

每隔一段，对照第二部分的四台机器查一次：
- 我是不是只奖励"答对"、很少奖励"提问"？（**标准答案机器** → 伤好奇/思辨）
- 即时刺激（屏幕/零食/不停的奖励）是不是太唾手可得？（**多巴胺劫持** → 伤专注/自控）
- 我是不是常把"困难"误当"危险"，替他扫平太多？（**包办型瘫痪** → 伤自主/韧性）
- 我驱动他的，是"想"多，还是"怕"多？（**恐惧驱动** → 伤内在动机）
